## OpenSearch k-NN 검색 마이그레이션 조사 및 문제 해결

### 작업 개요
기존 RDB 기반 키워드 검색을 OpenSearch k-NN 시맨틱 검색으로 마이그레이션하는 작업 진행. 기존 `unified-patents-v1` 인덱스(4.8M 문서, BGE-M3 임베딩)를 활용하여 검색 기능 개선 시도.

### 완료된 작업

#### 1. MCP 서비스 환경 설정 및 의존성 해결
- **설정 파일 업데이트**
  - `.env`: `OPENSEARCH_PATENT_INDEX=unified-patents-v1` 추가
  - `mcp/config/settings.py`: OpenSearch 인덱스 설정 추가
  - `mcp/database.py`: `AsyncOpenSearch` 클라이언트로 전환

- **의존성 충돌 해결**
  - `mcp/pyproject.toml` 업데이트:
    - `transformers >= 4.45.0`
    - `sentence-transformers >= 3.0.0`
    - `peft >= 0.13.0`
    - `accelerate >= 1.2.0`
    - `sentencepiece >= 0.2.0` 추가
  - `FlagEmbedding` 제거 (불안정성 문제)

#### 2. 임베딩 서비스 리팩토링
- **파일**: `mcp/services/embedding_service.py`
- `FlagEmbedding`에서 `sentence-transformers`로 전환
- BGE-M3 모델 안정적 로딩 및 임베딩 생성 확인 (1024 차원)
- 비동기 처리 지원

#### 3. 검색 쿼리 문제 진단 및 해결

**문제 발견**:
- 초기 시맨틱 검색이 0건 결과 반환
- 표준 k-NN 쿼리 `{"query": {"knn": {...}}}` 작동 안 함

**조사 과정**:
1. OpenSearch 연결 확인 (✅ 정상)
2. 인덱스 문서 수 확인 (✅ 4.8M+ 문서 존재)
3. 임베딩 생성 확인 (✅ 1024 차원 정상 생성)
4. 다양한 쿼리 형식 테스트

**테스트 결과**:

| 쿼리 타입 | 결과 | 성능 |
|----------|------|------|
| `{"query": {"knn": {...}}}` | ❌ 0건 | N/A |
| `{"query": {"match_all": {}}}` | ✅ 정상 | 빠름 |
| `script_score` with `knn_score` | ✅ 정상 (score: 0.9798961) | 느림 (60초+) |

**근본 원인 발견**:
- `unified-patents-v1` 인덱스에 `index.knn: true` 설정 누락
- 인덱스는 `knn_vector` 필드 타입 보유하지만 HNSW 그래프 구조 미생성
- 근사 k-NN 쿼리 실패, brute-force `script_score`만 작동
- 4.8M 문서에 대한 brute-force 검색으로 타임아웃 발생

### 기술적 발견 사항

#### OpenSearch 환경
- **버전**: OpenSearch 2.16.0
- **k-NN 플러그인**: 2.16.0.0
- **인덱스 설정**:
  - Field type: `knn_vector`
  - Dimension: 1024
  - Engine: `nmslib`
  - Space type: `cosinesimil`
  - Algorithm: HNSW (ef_construction=128, m=16)
  - **문제**: `index.knn: true` 설정 누락

#### 쿼리 형식 비교

**작동하는 쿼리** (하지만 느림):
```json
{
  "size": 3,
  "query": {
    "script_score": {
      "query": {"match_all": {}},
      "script": {
        "source": "knn_score",
        "lang": "knn",
        "params": {
          "field": "embedding",
          "query_value": [/* 1024 dim vector */],
          "space_type": "cosinesimil"
        }
      }
    }
  }
}
```

**작동하지 않는 쿼리** (index.knn: true 필요):
```json
{
  "size": 3,
  "query": {
    "knn": {
      "embedding": {
        "vector": [/* 1024 dim vector */],
        "k": 3
      }
    }
  }
}
```
