# Patent Board - Landing Page

## 🎯 새로운 랜딩 페이지 구현

사용자가 로그인하기 전에 플랫폼의 기능을 이해할 수 있도록 전문적인 소개 페이지가 추가되었습니다.

## ✨ 주요 기능

### 1. 히로 섹션 (Hero Section)
- **플랫폼 소개**: "AI-powered patent analysis platform" 명확한 제목과 설명
- **시각적 요소**: 그라데이션 배경과 패턴 아이콘
- **행동 유도 버튼**: "Get Started Free" 및 "Learn More" 버튼

### 2. 기능 소개 섹션 (Features)
- **AI-Powered Chat**: 자연어 인터페이스로 특허 쿼리 처리
- **Advanced Analytics**: 시장 분석 및 경쟁정보 제공
- **Automated Reports**: 상세한 특허 분석 보고서 자동 생성
- **Multi-Agent AI**: LangGraph 기반 멀티 에이전트 워크플로우
- **Database Integration**: MariaDB + Neo4j 통합
- **Real-time Processing**: 실시간 빠른 처리 속도

### 3. 작동 방식 (How It Works)
1. **Ask Questions**: 자연어로 특허 관련 질문
2. **AI Analysis**: AI 시스템이 데이터베이스 분석
3. **Get Insights**: 포괄적인 분석 결과 제공

### 4. 기술 스택 (Technology Stack)
- **LangGraph**: 멀티 에이전트 AI 워크플로우
- **OpenAI GPT**: 고급 언어 모델
- **FastAPI**: 고성능 백엔드
- **React**: 모던 프론트엔드 인터페이스
- **Neo4j**: 관계형 그래프 데이터베이스
- **MariaDB**: 구조화된 데이터 저장
- **Tailwind CSS**: 반응형 디자인 시스템
- **Vite**: 빠른 개발 빌드

### 5. 최종 행동 유도 (Call to Action)
- **Start Your Free Analysis**: 무료 분석 시작
- **View Documentation**: 기술 문서 확인
